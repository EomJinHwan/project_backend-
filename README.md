## 프로젝트 소개
스터디카페 서비스의 회원 인증과 사용자 관리를 담당하는 Node.js 기반 백엔드 서버입니다.  
팀 프로젝트에서 백엔드 인증 기능 구현을 담당했으며, 회원가입, 로그인, 아이디/비밀번호 찾기, SMS 인증, 네이버 OAuth 로그인 기능을 구현했습니다.

프로젝트는 AWS 기반 환경에 배포되었고, Gabia에서 구매한 도메인을 연결해 실제 서비스 접속까지 확인했습니다.  
다만 현재 저장소에는 제가 담당했던 백엔드 기능 중심의 코드만 정리되어 있습니다.

## 담당 역할
- Express 기반 회원 인증 API 구현
- 회원가입, 로그인, 아이디 중복 확인, 아이디/비밀번호 찾기 기능 구현
- bcrypt를 활용한 비밀번호 암호화 처리
- MySQL 기반 사용자 데이터 조회 및 저장 로직 구현
- CoolSMS 기반 SMS 인증번호 발송 및 검증 기능 구현
- 네이버 OAuth 로그인 기능 구현
- 팀 프로젝트 배포 과정에서 AWS 기반 서버 운영 흐름과 Gabia 도메인 연결 구조 경험

## 주요 기능

### 인증 및 회원 관리
- 회원가입, 로그인, 아이디 중복 확인, 아이디/비밀번호 찾기 기능 구현
- bcrypt를 활용한 비밀번호 암호화 처리
- 로그인 성공 시 로그인 이력 저장

### 외부 API 연동
- CoolSMS를 활용한 SMS 인증번호 발송 및 검증 기능 구현
- 네이버 OAuth API를 활용한 소셜 로그인 기능 구현

### 서버 구조
- Express 기반 API 서버 구성
- Router / Controller / Service 계층 분리
- MySQL 기반 사용자 데이터 조회 및 저장 로직 구현

### 배포 경험
- 팀 프로젝트에서 AWS 기반 배포 환경을 경험했습니다.
- Gabia에서 구매한 도메인을 연결해 실제 서비스 접속까지 확인했습니다.
- 배포 설정의 주 담당자는 팀원이었으며, 저는 백엔드 기능 구현과 API 연동을 중심으로 담당했습니다.
